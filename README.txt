WELCOME TO GROUP 17'S COURSE AND PROGRAM MANAGEMENT PROGRAM
-------------------------------------------------------------

To run the program, please do the following

1. Unzip file to your desired destination
2. Double-Click the Run.jar file or run it from terminal
3. When it first starts, please make an account by clicking "New Faculty?"
4. Once you are registered, login to explore the application
5. For "Edit" forms, make sure you have the selected the course, program, or application that
   you wish to edit and click the "edit" button to make changes to existing data
6. Aside from that, the application is very straight forward
7. Have fun

Note: You may have the resize the frame during the application
